# 朱雀抽奖 MoviePilot 插件

自动抽取朱雀站点喇叭球，显示灵石余额，支持定时任务和消息推送。

## 功能

1. **自动抽奖**：定时抽取朱雀站点喇叭球
2. **灵石余额**：实时显示朱雀灵石余额
3. **余额对比**：抽奖前后余额对比
4. **消息推送**：抽奖结果推送到微信/Telegram
5. **定时任务**：支持 Cron 表达式定时抽奖
6. **灵石提醒**：低于设定阈值时提醒

## 安装

### 方法 1：从 GitHub 安装（推荐）

1. 打开 MoviePilot 插件管理
2. 点击"从 GitHub 安装"
3. 输入仓库 URL：`https://github.com/hotluqibo/zhuque-lottery-moviepilot-plugin`
4. 点击"安装"
5. 等待安装完成

### 方法 2：手动安装

1. 下载插件代码：`__init__.py`
2. 打开 MoviePilot 插件管理
3. 点击"本地安装"
4. 上传 `__init__.py` 文件
5. 等待安装完成

## 配置

安装后，需要配置以下参数：

| 配置项 | 说明 | 获取方法 |
|--------|------|----------|
| ✅ 启用插件 | 开启/关闭插件 | 开关 |
| 🍪 朱雀 Cookie | 登录朱雀后的 Cookie | 见下方 |
| 🔑 CSRF Token | 从朱雀页面获取 | 见下方 |
| ⏰ 定时任务 | Cron 表达式 | `0 8 * * *`（每天早上8点） |
| 💰 灵石阈值 | 低于此值时提醒 | `1000.0` |
| 📢 开启通知 | 抽奖后发送通知 | 开关 |

## 获取 Cookie 和 CSRF Token

### 方法 A：使用浏览器抓包（推荐）

1. 打开浏览器，登录朱雀站点：`https://zhuque.in`
2. 按 `F12` 打开开发者工具
3. 切换到 **Network** 标签
4. 点击 **"抽奖"** 按钮
5. 查看刚才的请求，找到：
   - **Cookie**：`socute=s%3A...`
   - **x-csrf-token**：`a83f6507da82cab683a74275e0ee1f3a`

### 方法 B：从浏览器 Cookie 中获取

1. 打开浏览器，登录朱雀站点
2. 按 `F12` → **Application** → **Cookies** → `https://zhuque.in`
3. 找到 `socute` 的值，复制到 **朱雀 Cookie** 配置项
4. 刷新页面，查看页面源代码，找到 `csrf-token` 的值，复制到 **CSRF Token** 配置项

## 使用方法

1. 启用插件
2. 配置 Cookie 和 CSRF Token
3. 保存配置
4. 点击"立即抽奖"测试
5. 设置定时任务（可选）

## 注意事项

- Cookie 和 CSRF Token 会过期，需要定期更新
- 每次抽奖花费 4 灵石
- 建议设置灵石阈值提醒
